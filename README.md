# FM-80 Synthesizer

ブラウザで動作する80年代風 FM シンセサイザー＋ステップシーケンサー。Web Audio API のみで実装。

- **FMシンセ**（2オペレーター、ポリフォニック）を鍵盤で演奏
- **TB303風ステップシーケンサー**を2系統（SEQ 1 / SEQ 2）搭載。ベースラインをリアルタイムまたはステップ入力で組める
- **ドラムシーケンサー**（4トラック × 16ステップ）が SEQ 1 と同期

---

## 起動

`index.html` をブラウザで開き、**POWER ON** をタップ。
（Web Audio API の制約上、ユーザー操作が必要）

---

## 機能

### 1. ヘッダー

- **FM-80 ロゴ** の下に 2 つの LED インジケーターを配置
  - 左 LED（緑）: 電源オン時に点灯
  - 右 LED（緑）: 鍵盤を弾いている間点灯
- **プリセットボタン**（8 種）をロゴ右側に配置

| プリセット | 特徴 |
|-----------|------|
| E.PIANO   | 電子ピアノ（デフォルト） |
| BRASS     | ブラス |
| STRINGS   | ストリングス |
| BASS      | ベース |
| BELL      | ベル |
| MARIMBA   | マリンバ |
| LEAD      | リードシンセ |
| ORGAN     | オルガン |

---

### 2. パラメーターパネル（ノブ操作）

上下ドラッグでノブを回す。ダブルクリックでデフォルト値にリセット。

**ENV（エンベロープ）**

| ノブ | 範囲 | 説明 |
|-----|------|------|
| ATK | 0.001〜2 秒 | アタックタイム |
| DCY | 0.01〜3 秒 | ディケイタイム |
| SUS | 0〜1 | サスティンレベル |
| REL | 0.01〜5 秒 | リリースタイム |

**FM（変調）**

| ノブ | 範囲 | 説明 |
|-----|------|------|
| RATIO | 0.5〜16 | キャリアに対するモジュレーター周波数比 |
| INDEX | 0〜12 | FM変調深度（倍音量） |
| ENV.D | 0.01〜3 秒 | モジュレーターエンベロープのディケイ |

**EFFECTS**（鍵盤音色のみに適用）

| ノブ | 範囲 | 説明 |
|-----|------|------|
| DIST   | 0〜1 | ディストーション（tanh サチュレーション） |
| DELAY  | 0〜1 秒 | ディレイタイム（0 でバイパス） |
| FB     | 0〜0.92 | ディレイフィードバック量 |
| REVERB | 0〜1 | リバーブ（マルチタップディレイ）の深さ |

---

### 3. ステップシーケンサー（SEQ 1 / SEQ 2）

TB303 風のベースライン専用シーケンサーを 2 系統搭載。それぞれ独立した TB303Synth 音源を持ち、互いに干渉しない。
16 ステップ、1 ループ = 1 小節（4/4 拍子）。PLAY で両シーケンサーが同時に再生・停止する。

#### 基本操作

| 操作 | 説明 |
|-----|------|
| ▲ / ▼ | 各ステップのノート（MIDI 24〜72）を半音単位で変更 |
| ON / — | ステップの ON / OFF を切り替え |
| − / + BPM | BPM を 5 刻みで変更（40〜240 BPM）。両シーケンサーに同期 |
| ▶ PLAY | SEQ 1・SEQ 2 を同時に再生開始 |
| ■ STOP | 両シーケンサーを停止 |

#### ステップ書き込みモード（STEP）

鍵盤を使って 1 ステップずつノートを手動入力するモード。

1. **STEP** ボタンを押してステップ書き込みモードをオンにする（ボタンが点灯、カーソルがステップ 1 に移動）
2. 鍵盤（画面 or PC キーボード）を弾くと、**カーソル位置のステップにそのノートが記録され、ON になる**
3. カーソルは自動で次のステップへ進む
4. 無音ステップ（OFF）を入力したい場合は **REST** ボタンを押す
5. ステップセルをタップするとカーソルをそのステップに直接移動できる
6. 入力が完了したら再度 **STEP** ボタンを押してモードをオフにする

> **Note:** STEP モードは SEQ 1 と SEQ 2 のどちらか一方しか同時に有効にできない。もう一方の STEP を押すと自動で切り替わる。

---

### 4. 鍵盤

4 オクターブ表示。横スクロール対応。

| 操作 | 説明 |
|-----|------|
| タップ／クリック | 発音 |
| − / + OCT | 表示オクターブを変更（OCT 2〜7） |
| BEND スライダー | ピッチベンド（±2 半音） |

**PC キーボード対応（デスクトップ）**

```
白鍵: A S D F G H J K L
黒鍵: W E   T Y U   O P
```

現在オクターブ基準。押しっぱなしリピートなし。

---

### 5. ドラムトラック

16 ステップのドラムシーケンサー。SEQ 1 と同期して再生。

| トラック | 音色 |
|---------|------|
| KICK  | 正弦波ピッチダウン（120 Hz → 30 Hz） |
| SNARE | ノイズ + トーン（180 Hz）の合成 |
| H.HAT | ハイパスフィルター掛けノイズ（8 kHz） |
| CLAP  | 3 連バースト × バンドパスノイズ |

- ヘッダーの **DRUMS OFF / ON** ボタンでドラムトラック全体をミュート
- ボリュームは右スワイプメニューの **DRUM** スライダーで調整

---

### 6. 右スワイプメニュー

画面右端からスワイプして開く。ボリューム調整と SEQ エフェクトが 1 つのドロワーに統合されている。

| 操作 | 説明 |
|-----|------|
| 画面右端から左スワイプ（60 px 以上） | ドロワーを開く |
| 右スワイプ or バックドロップをタップ | ドロワーを閉じる |

#### 上段：VOLUME

各音源のボリュームを個別に調整する縦スライダー。

| スライダー | 対象 | デフォルト |
|-----------|------|----------|
| SYNTH | FMシンセ（鍵盤） | 0.6 |
| SEQ   | TB303シーケンサー（SEQ 1・SEQ 2 共通） | 0.55 |
| DRUM  | ドラムトラック | 0.7 |

#### 下段：SEQ FX

ステップシーケンサー（SEQ 1）専用のエフェクト。

| スライダー | 効果 | デフォルト |
|-----------|------|----------|
| DIST   | ディストーション（tanh サチュレーション） | 0 |
| COMP   | コンプレッサー（threshold 0〜−40 dB、ratio 1〜16） | 0 |
| DELAY  | ディレイタイム（0〜1 秒） | 0 |
| REVERB | リバーブミックス（マルチタップディレイ） | 0.06 |

## 音声アーキテクチャ

```
[FMSynth]（ポリフォニック）
  Mod Osc ──→ Mod Env ──→ Carrier 周波数変調
  Carrier Osc ──→ Carrier Env ──→ Filter (LPF 8kHz)
    └──→ WaveShaper (Distortion)
           ├──→ MasterGain ──→ 出力（ドライ）
           ├──→ DelaySend ──→ DelayNode ──→ DelayWet ──→ MasterGain
           │                      ↑ DelayFeedback ┘
           └──→ ReverbSend ──→ マルチタップディレイ（5タップ）──→ MasterGain

[TB303Synth × 2]（モノフォニック、SEQ 1 / SEQ 2 それぞれ独立）
  Sawtooth Osc ──→ Resonant LPF (Q=8、ノート追従カットオフ) ──→ AmpEnv
    └──→ WaveShaper (Distortion)
           └──→ DynamicsCompressor
                  ├──→ MasterGain ──→ 出力（ドライ）
                  ├──→ DelaySend ──→ DelayNode ──→ DelayWet ──→ MasterGain
                  │                      ↑ DelayFeedback ┘
                  └──→ ReverbSend ──→ マルチタップディレイ（3タップ）──→ MasterGain

[DrumSynth]
  各打楽器 ──→ MasterGain ──→ 出力
```

### 各シンセの特徴

| 項目 | FMSynth | TB303Synth | DrumSynth |
|------|---------|-----------|-----------|
| 発音方式 | ポリフォニック | モノフォニック | 1ショット |
| 波形 | サイン波（FM合成） | ノコギリ波 | 合成（各楽器固有） |
| フィルター | LPF 8kHz / Q=1.5 | LPF ノート追従 / Q=8 | BP / HP（楽器別） |
| ディストーション | ✓ | ✓ | — |
| コンプレッサー | — | ✓ | — |
| ディレイ | ✓（2.0s max） | ✓（2.0s max） | — |
| リバーブ | ✓（5タップ） | ✓（3タップ） | — |

- リバーブは `ConvolverNode` を使わず、マルチタップディレイで実装（iOS 互換）
