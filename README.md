# FM-80 Synthesizer

ブラウザで動作する80年代風 FM シンセサイザー＋マルチトラックシーケンサー。Web Audio API のみで実装。インストール不要。

---

## 起動

`index.html` をブラウザで開き、**POWER ON** をタップ。
（Web Audio API の制約上、ユーザー操作が必要）

---

## 構成ファイル

```
index.html   — UI マークアップ
style.css    — 80年代ダークテーマ
synth.js     — FMSynth / TB303Synth クラス
sequencer.js — Sequencer クラス + UI 初期化
chord.js     — ChordSequencer クラス
drum.js      — DrumSynth クラス + ドラムグリッド UI
melody.js    — MelodyTrack クラス（録音・再生）
ui.js        — 鍵盤・ノブ・各種コントローラー
```

## 機能一覧

### 1. ヘッダー

- **⚙ ボタン**（左上）: VOL / FX ドロワーを開閉
- **LED**: 左=電源オン、右=発音中
- **プリセットボタン**（8種）: 鍵盤シンセの音色を切り替え

| プリセット | 特徴 |
|-----------|------|
| E.PIANO   | 電子ピアノ（デフォルト） |
| BRASS     | ブラス |
| STRINGS   | ストリングス |
| BASS      | ベース |
| BELL      | ベル |
| MARIMBA   | マリンバ |
| LEAD      | リードシンセ |
| ORGAN     | オルガン |

---

### 2. パラメーターパネル（ノブ）

上下ドラッグで調整。ダブルクリックでデフォルト値にリセット。

**ENV（エンベロープ）**

| ノブ | 範囲 | 説明 |
|-----|------|------|
| ATK | 0.001〜2 秒 | アタックタイム |
| DCY | 0.01〜3 秒 | ディケイタイム |
| SUS | 0〜1 | サスティンレベル |
| REL | 0.01〜5 秒 | リリースタイム |

**FM（変調）**

| ノブ | 範囲 | 説明 |
|-----|------|------|
| RATIO | 0.5〜16 | モジュレーター周波数比 |
| INDEX | 0〜12 | FM 変調深度（倍音量） |
| ENV.D | 0.01〜3 秒 | モジュレーターエンベロープのディケイ |

**EFFECTS**（鍵盤音色に適用）

| ノブ | 範囲 | 説明 |
|-----|------|------|
| DIST   | 0〜1 | ディストーション |
| DELAY  | 0〜1 秒 | ディレイタイム |
| FB     | 0〜0.92 | ディレイフィードバック |
| REVERB | 0〜1 | リバーブの深さ |

---

### 3. ステップシーケンサー（SEQ 1 / SEQ 2）

TB303 風のベースライン専用シーケンサーを 2 系統搭載。16 ステップ、1 ループ = 1 小節（4/4）。
SEQ 2 はヘッダーの **SEQ 2 ▼** ボタンで折りたたみ可能。

#### 操作

| 操作 | 説明 |
|-----|------|
| ▲ / ▼ | ステップのノートを半音単位で変更 |
| ON / — | ステップの ON / OFF |
| − / + BPM | BPM を変更（40〜240）。両 SEQ に同期 |
| **STEP** | ステップ書き込みモード（鍵盤 → ステップ順に入力） |
| **REST** | 現在ステップを無音にして次へ進む |
| **⏺ REC** | リアルタイム録音（4拍カウントダウン後に開始） |
| **▶ PLAY** | 全シーケンサーを一括で再生 / 停止 |

#### パターンバンク（8スロット、全バンク連動）

**SAVE** → スロット（P1〜P8）を選択で保存。スロットを選択するだけでロード。
再生中にスロットを選ぶと、現在ループの末尾で自動切り替え（ピンク点滅 = 待機中）。

> P1〜P8 はすべてのシーケンサー（SEQ1・SEQ2・CHORD・MELODY）で共有される。
> どのバンクのスロットを押しても全バンクが同じスロットに切り替わる。

---

### 4. コードシーケンサー（CHORD）

**CHORD ▼** ボタンで展開。FMSynth（STRINGSプリセット）を使ってコードを順に鳴らす。

| 操作 | 説明 |
|-----|------|
| コードタイプ選択 | MAJ / MIN / 7th / m7 / M7 / DIM / SUS |
| 根音パレット | C〜B の 12 音を押してステップに割り当て |
| − / + OCT | コードのオクターブ変更 |
| **STEP** | ステップ書き込みモード |

パターンバンク（8スロット）あり。SEQ1・SEQ2 と連動。

---

### 5. 鍵盤

4 オクターブ表示（横スクロール対応）。

| 操作 | 説明 |
|-----|------|
| タップ / クリック | 発音 |
| − / + OCT | 表示オクターブを変更（OCT 2〜7） |
| BEND スライダー | ピッチベンド（±2 半音） |

**PC キーボード**

```
白鍵: A S D F G H J K L
黒鍵: W E   T Y U   O P
```

---

### 6. メロディトラック（MELODY）

鍵盤で弾いた音をタイムスタンプ付きで録音し、シーケンサーのループに合わせてそのまま再生する。
複数音（和音・速いパッセージ）もそのまま録音・再生可能。

#### 操作

| ボタン | 説明 |
|-------|------|
| **ON / OFF** | 再生の有効 / 無効 |
| **⏺ REC** | 次のループ先頭から録音開始（1ループで自動停止） |
| **▶ PLAY** | 全シーケンサーを一括で再生 / 停止 |

#### タイムライン表示

録音した音符の位置と長さを横棒で可視化。縦の区切り線は 4 拍の境界。
- 再生中: シアンのカーソルが流れる
- 録音中: 赤のカーソルが流れ、音符が録音と同時に表示される

#### パターンバンク（8スロット）

SEQ1・SEQ2・CHORD と連動。スロットには音符データと音色（プリセット）を保存。
ロード時に音色も自動復元される。

---

### 7. ドラムトラック（DRUMS）

16 ステップ × 6 トラックのドラムシーケンサー。SEQ 1 と同期。

| トラック | 音色 |
|---------|------|
| KICK    | 808 風ピッチダウン（160 Hz → 35 Hz） |
| SNARE   | ノイズ + トーン合成 |
| H.HAT   | ハイパスノイズ（8 kHz） |
| O.HAT   | オープンハイハット（やや長め） |
| CLAP    | 3 連バーストノイズ |
| COWBELL | 2 倍音デチューン矩形波 |

デフォルトパターン: ダフトパンク風 4 つ打ち（KICK 全拍 / SNARE 2・4 拍 / H.HAT 8 分 / O.HAT 裏拍 / CLAP 2・4 拍）

ヘッダーの **ON / OFF** ボタンでドラム全体をミュート。

---

### 8. VOL / FX ドロワー

#### 開き方

| 操作 | 説明 |
|-----|------|
| **⚙** ボタン（ヘッダー） | クリックでトグル |
| `→`（右矢印キー）/ `\`（バックスラッシュ） | 開く |
| `←`（左矢印キー）/ `Escape` | 閉じる |
| 画面右端から左スワイプ（タッチ） | 開く |
| バックドロップをクリック / 右スワイプ | 閉じる |

#### VOLUME

| スライダー | 対象 |
|-----------|------|
| SYNTH  | FMシンセ（鍵盤・メロディ） |
| SEQ1   | TB303 ベースシンセ 1 |
| SEQ2   | TB303 ベースシンセ 2 |
| CHORD  | コードシンセ |
| DRUM   | ドラムトラック |

#### SEQ FX（SEQ 1 専用）

| スライダー | 効果 |
|-----------|------|
| DIST   | ディストーション |
| COMP   | コンプレッサー |
| DELAY  | ディレイタイム |
| REVERB | リバーブミックス |

#### EXPORT（WAV 書き出し）

全音源をミックスしてリアルタイム録音。**⏺ REC** → **⏹ STOP** → **⬇ WAV** でダウンロード。
出力: 16-bit PCM ステレオ WAV。

## 音声アーキテクチャ

```
[FMSynth]（ポリフォニック）鍵盤演奏 + メロディ再生
  Mod Osc ──→ Mod Env ──→ Carrier 周波数変調
  Carrier Osc ──→ Carrier Env ──→ Filter (LPF 8kHz)
    └──→ WaveShaper (Distortion)
           ├──→ MasterGain → 出力
           ├──→ DelaySend  → DelayNode (2s max) → MasterGain
           └──→ ReverbSend → マルチタップディレイ (5タップ) → MasterGain

[TB303Synth × 2]（モノフォニック、SEQ 1 / SEQ 2）
  Sawtooth Osc ──→ Resonant LPF (Q=8、ノート追従) ──→ AmpEnv
    └──→ WaveShaper → DynamicsCompressor
           ├──→ MasterGain → 出力
           ├──→ DelaySend  → DelayNode → MasterGain
           └──→ ReverbSend → マルチタップディレイ (3タップ) → MasterGain

[FMSynth]（ポリフォニック）コードシーケンサー専用（STRINGSプリセット固定）

[DrumSynth]
  KICK    : 正弦波 + ピッチエンベロープ + クリックトランジェント
  SNARE   : バンドパスノイズ + トーン + 高域スナップ
  H.HAT   : ハイパスノイズ（短）
  O.HAT   : ハイパスノイズ（長）
  CLAP    : 3連バーストバンドパスノイズ
  COWBELL : デチューン矩形波 2音 + バンドパス
  → サイドチェイン圧縮 (KICK → TB303 × 2)
```
