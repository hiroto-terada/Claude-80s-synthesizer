# FM-80 Synthesizer

ブラウザで動作する80年代風FMシンセサイザー。Web Audio API のみで実装。

---

## 起動

`index.html` をブラウザで開き、**POWER ON** をタップ。
（Web Audio API の制約上、ユーザー操作が必要）

---

## 構成ファイル

```
index.html   — UI マークアップ
style.css    — 80年代ダークテーマ
synth.js     — FMSynth / TB303Synth クラス
sequencer.js — Sequencer クラス + UI 初期化
ui.js        — 鍵盤・ノブ・各種コントローラー
```

## 機能

### 1. プリセット（8種）

ヘッダーのボタンで音色を即時切り替え。

| プリセット | 特徴 |
|-----------|------|
| E.PIANO   | 電子ピアノ（デフォルト） |
| BRASS     | ブラス |
| STRINGS   | ストリングス |
| BASS      | ベース |
| BELL      | ベル |
| MARIMBA   | マリンバ |
| LEAD      | リードシンセ |
| ORGAN     | オルガン |

---

### 2. パラメーターパネル（ノブ操作）

上下ドラッグでノブを回す。1行に3セクション表示。

**ENV（エンベロープ）**

| ノブ | 範囲 | 説明 |
|-----|------|------|
| ATK | 0.001〜2 秒 | アタックタイム |
| DCY | 0.01〜3 秒 | ディケイタイム |
| SUS | 0〜1 | サスティンレベル |
| REL | 0.01〜5 秒 | リリースタイム |

**FM（変調）**

| ノブ | 範囲 | 説明 |
|-----|------|------|
| RATIO | 0.5〜16 | キャリアに対するモジュレーター周波数比 |
| INDEX | 0〜12 | FM変調深度（音の倍音量） |
| ENV.D | 0.01〜3 秒 | モジュレーターエンベロープのディケイ |

**EFFECTS**

| ノブ | 範囲 | 説明 |
|-----|------|------|
| FILTER | 200〜18000 Hz | ローパスフィルターのカットオフ |
| REVERB | 0〜1 | リバーブ（マルチタップディレイ）の深さ |
| VOL    | 0〜1 | FMシンセのマスターボリューム |

---

### 3. ステップシーケンサー

**TB303 風のベースライン専用**（FMシンセとは独立した音源）。16ステップ、1ループ = 1小節（4/4拍子）。

#### 基本操作

| 操作 | 説明 |
|-----|------|
| ▲ / ▼ | 各ステップのノート（MIDI 24〜72）を半音単位で変更 |
| ON / — | ステップのON/OFFを切り替え |
| − / + BPM | BPMを5刻みで変更（40〜240 BPM） |
| ▶ PLAY | 再生開始 |
| ■ STOP | 停止 |

#### リアルタイム録音モード（⏺ REC）

1. **⏺ REC** を押す
2. 4拍カウントダウン開始（クリック音 + ボタンに `4... 3... 2... 1...` 表示）
3. カウント終了後、録音スタート
4. 鍵盤で弾いた音が **テンポに合わせてステップに自動記録**
   - 各ステップ中に最後に押した音がそのステップに記録される
   - 何も押さなかったステップは無音（OFF）になる
5. 16ステップ分（1ループ）完了したら **自動でREC終了 → 再生継続**

録音中はクリック音（メトロノーム）が鳴り続ける。
- 拍頭（ステップ 1, 5, 9, 13）: 高音アクセント（1400 Hz）
- その他の拍: 通常クリック（900 Hz）

---

### 4. 鍵盤

4オクターブ表示。横スクロール対応。

| 操作 | 説明 |
|-----|------|
| タップ／クリック | 発音 |
| − / + OCT | 表示オクターブを変更（OCT 2〜7） |
| BEND スライダー | ピッチベンド（±2半音） |

**PCキーボード対応（デスクトップ）**

```
白鍵: A S D F G H J K L
黒鍵: W E   T Y U   O P
```
現在オクターブ基準。押しっぱなしリピートなし。

---

### 5. ボリュームドロワー（モバイル向け）

シンセとシーケンサーのボリュームを個別に調整できる。

| 操作 | 説明 |
|-----|------|
| 画面左端から右スワイプ（60px 以上） | ドロワーを開く |
| 左スワイプ or バックドロップタップ | ドロワーを閉じる |

ドロワー内の縦スライダー:
- **SYNTH**: FMシンセ（鍵盤）のボリューム（デフォルト: 0.6）
- **SEQ**: TB303シーケンサーのボリューム（デフォルト: 0.55）

## 音声アーキテクチャ

```
[FMSynth]
  Carrier Osc ──→ Carrier Env ──→ Filter ──→ MasterGain ──→ 出力
  Mod Osc ──→ Mod Env ──→ Carrier 周波数（FM変調）
  Filter ──→ ReverbSend ──→ マルチタップディレイ（5タップ）──→ MasterGain

[TB303Synth]
  Sawtooth Osc ──→ Resonant Filter ──→ AmpEnv ──→ MasterGain ──→ 出力
  AmpEnv ──→ ReverbSend ──→ マルチタップディレイ ──→ MasterGain
```

- **FMSynth**: ポリフォニック（ノートごとに独立したオシレーター）
- **TB303Synth**: モノフォニック（シーケンサー専用）
- リバーブは ConvolverNode を使わず、マルチタップディレイで実装（iOS 互換）

## 動作環境

- モダンブラウザ（Web Audio API 対応）
- iOS Safari: ユーザー操作後に AudioContext を自動 resume
- サーバー不要（静的ファイルのみ）
